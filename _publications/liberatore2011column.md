abstract: >
  We present a column generation algorithm for the vehicle routing problem with
  soft time windows, where time window violations are penalized. The pricing
  subproblem is solved with an efficient dynamic programming algorithm, and the
  method is embedded in a branch-and-price framework to obtain provably optimal
  solutions.
